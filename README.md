# LevelNotes — working version

This ZIP contains:

- `index.html`
- `styles.css`
- `script.js`

## Working functionality

- Clicking any large subject card opens the matching syllabus panel.
- Clicking any left-hand subject tab changes the visible topic panel.
- Clicking a topic applies the subject/topic filter and scrolls to the notes.
- Subject, syllabus, level and resource filters work.
- Search works.
- Save-heart buttons work.
- Dark mode works and is remembered.

## Open it

Double-click `index.html`, or run:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
